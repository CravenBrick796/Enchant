⚙ Principes — Enclume optimale

  • Règle n°1 : combiner les livres par paires avant de les appliquer sur l'objet
  • Règle n°2 : l'enchantement le plus cher s'applique en premier (pénalité 0)
  • Limite : toute opération > 39 niveaux → « Trop cher ! »
  • 📚 violet = combiner deux livres · ✔ vert = appliquer sur l'objet

  • Aura de feuII
  • ReculII
  • AffilageIII
  • SoliditéIII

Ordre optimal — 7 enchantements
1📚 Butin III + 📚 Affilage III → Livre A (6+6=12)
2📚 Tranchant V + 📚 Aura de feu II → Livre B (5+4=9)
3📚 Solidité III + 📚 Raccommodage → Livre C (3+2=5)
— application sur l'épée —
4⚔ Épée + Livre A (Butin + Affilage, coût 12)
5⚔ Épée + Livre B (Tranchant + Aura de feu, coût 9+2pen=11)
6⚔ Épée + Livre C (Solidité + Raccommodage, coût 5+4pen=9)
7⚔ Épée + 📚 Recul II (coût 2+8pen=10)
Total : ~43 niveaux (toutes opérations < 39 ✓)
⚠ Tranchant / Fléau des arthropodes / Châtiment incompatibles. Affilage = de retour depuis la 1.20.5. Aura de feu = "Fire Aspect" en anglais.
